# 🚀 AI DevOps Platform (RAG + MLOps + GenAI)

## 📌 Overview
End-to-end AI platform integrating DevOps, MLOps, and GenAI.

## ⚙️ Tech Stack
- AWS (EKS, S3)
- Kubernetes
- Terraform
- MLflow
- FAISS (RAG)
- Prometheus + Grafana
- LocalStack

## 🔄 Architecture
User → FastAPI → RAG → LLM → Response

## 🚀 Features
- CI/CD automation
- Kubernetes deployment
- RAG-based AI responses
- Observability with Prometheus
